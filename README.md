# websematica_datos_abiertos_MTI

## Proyecto final para la asignatura de Web Semantica y Datos Abiertos del programa del MTI

- Este repositorio contiene el trabajo final desarrollado para el Magíster en Tecnologías de la Información (MTI) de la Universidad Técnica Federico Santa María. El proyecto explora la intersección entre la Web Semántica, el uso de Grafos de Conocimiento (KG) y las capacidades de los Modelos de Lenguaje de Gran Escala (LLM).

- El trabajo se divide en cuatro pilares fundamentales que demuestran el ciclo de vida de los datos vinculados (Linked Data) y su impacto en la inteligencia artificial moderna:

1. Creación y Validación de Grafos (RDF)

2. Consultas y Comparativa: SPARQL vs. LLM

3. Sinergia Neuro-simbólica y GraphRAG

4. Privacidad y Proyecto Solid
